总结
1.git 常用命令： git verson查看git版本信息 git help：git帮助命令 2.管理项目常用命令： git init：初始化仓库 git add your filesname yourfilestype(比如：index.html...and so on...)提交单个文件 git add . :将所有文件和文件夹提交到文件的队列中 git commit：提交到版本库中 3.备份本地项目到云端： 1)注册登入云端git服务器 2)在云端建一个备份项目 3)克隆云端项目到本地 4)复制本地项目到git 项目中 5)cd项目目录 6)git add . 提交所有文件 7)gitcommit -m “描述” 8)git push origin master:提交到云端版本库中
